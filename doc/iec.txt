==============================================================================
INTRODUCTION                                        *iec-vim-introduction*

This plugin provides IEC 61131-3 languages support. Examples can be found in
samples directory.

==============================================================================
MATIEC INTEGRATION                                  *iec-vim-matiec*

Current plugin designed to pair with matiec compiler backend. Read more about
matiec: https://bitbucket.org/mjsousa/matiec.

To translate currently editing .st file to C sources use following command:

	:make %<

Available configuration variables:

    g:matiec_path       Path to local matiec installation
